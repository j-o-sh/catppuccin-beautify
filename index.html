Catppuccin Beautify

A beautify theme based on Catppuccin.

The emphasis is on a streamlined and clutterfree DOM where a good looking theme emerges from the HTML semantics. (So yes, it is basically an antithesis to Tailwind. Fight me! ðŸ˜ˆ)

Colors

The colors are the Catppuccin colors. (What a supprise, I know ðŸŽ‰).

They are grouped into themes that correspond to the Catppuccin themes and selected for your document in one of two ways:

  • When no theme class is given, the css respects the users color scheme preferences via `@media (prefers-color-scheme)`. It will select Latte for a light theme preference and Mocha for the dark theme preference.
  • You can manually set the theme used, be specifying a `data-catppuccin-flavour` attribute with either `latte`, `frappe`, `macchiato` or `mocha`.
